# ProfessionAI - Data Analysis Projects

Collection of projects developed during the **ProfessionAI Data Analysis Master**.
Each project covers different aspects of data analysis, from Python programming 
to machine learning and SQL.

---

## Python Projects

### Inventory Management System
Software gestionale per un negozio con funzionalità di aggiunta prodotti, 
registrazione vendite e calcolo profitti.

**Tecnologie:** Python (OOP, dizionari, gestione file)

### Aviation Accidents Analysis (1919-2023)
Analisi di quasi 25.000 incidenti aerei con identificazione di trend storici, 
nazioni e operatori più a rischio.

**Tecnologie:** Python (Pandas, Matplotlib)

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/EliaVenturini/professionai-projects/blob/main/Progetto%20Analisi%20dei%20disastri%20aerei%20dal%201919%20al%202023.ipynb)

### Spam Detection & NLP Analysis
Classificazione email SPAM tramite reti neurali (MLP), topic modeling (LDA), 
analisi della distanza semantica e NER per estrazione organizzazioni.

**Tecnologie:** Python (Pandas, Scikit-learn, Gensim, SpaCy, NLTK)

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/EliaVenturini/professionai-projects/blob/main/Progetto%20Spam%20Detection%20Natural%20Processing%20(VERIFICATO).ipynb)

### Credit Card Customer Segmentation
Segmentazione di 9.000 clienti in 7 cluster tramite KMeans e PCA. 
Modello Random Forest per classificazione e feature selection con RFECV. Accuracy: 98%.

**Tecnologie:** Python (Pandas, Scikit-learn, Seaborn, Matplotlib)

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/EliaVenturini/professionai-projects/blob/main/Progetto_of_credit_card_customers.ipynb)
